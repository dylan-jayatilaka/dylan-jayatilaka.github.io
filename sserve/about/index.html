    About Tonto

    Hi there.

    What is Tonto?

    Tonto is an open source code library for quantum crystallography. It is focussed on combining ab initio quantum chemistry for electronic structure with diffraction experiments on molecular crystals.

    Tonto’s main claim to fame is that it can

    • Refine hydrogen atom positions from X-ray diffraction experiments on small molecules, something previously thought to be impossible
    • Refine anharmonic vibration parameters for strong scattering atoms.
    • Produce “experimental wavefunctions”. From such wavefunctions one can derive reconstructed experimental information, including electrical moments, refractive indices, first and second order susceptibilities — even experimental bond indices.

    Tonto does all this by using tailor-made scattering factors for each atom, derived from a molecular wavefunction; or else it contrains a wave function to fit of the diffraction data.

    • Produce Hirshfeld surfaces, corresponding fingerprints and interactions energies, through an interface to the popular CrystalExplorer program widely used in the Crystal Engineering field.

    Is it free?

    Yes. Also, since the code is under the GPL2 license, you can clone and fork the repository and do what you like with it. The code is extremely well commented and is highly modular.

    What happened to the SourceForge site?

    Ah.

    It was getting harder to keep a functional system using the trusty subversion, so we have moved here to GitHub now that the momentum behind it seems unstoppable. Now doubt we will change in the future, just as we did from using no backups, to CVS, and to subversion.
